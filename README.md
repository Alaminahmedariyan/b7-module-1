## 📚 types-of-typescript — Topic তালিকা

### 1-4 Primitive Types
`string`, `number`, `boolean`, `null`, `undefined` — TypeScript-এর সবচেয়ে basic type।

### 1-5 Non-Primitive Types
`array`, `object`, `tuple` — একাধিক value ধরতে পারে এমন type।

### 1-6 Object, Literal & Optional Type
Object-এর shape, নির্দিষ্ট value (`"left" | "right"`), এবং optional property (`?`)।

### 1-7 Function in TypeScript
Parameter এবং return type সহ function লেখা।

### 1-8 Rest & Spread Operator
`...` দিয়ে অনেক argument নেওয়া এবং array/object ছড়িয়ে দেওয়া।

### 1-9 Destructuring
Object বা array থেকে সরাসরি value বের করে নেওয়া।

### 1-10 Type Alias
`type` keyword দিয়ে type-কে নাম দেওয়া।

### 1-11 Union & Intersection Types
`|` দিয়ে যেকোনো একটি, `&` দিয়ে দুটো type একসাথে।

### 1-12 Ternary, Nullish Coalescing & Optional Chaining
`? :` এক লাইনে if-else, `??` null হলে default, `?.` crash ছাড়া access।

### 1-13 Nullable, Unknown & Never Type
`null` allow করা, নিরাপদ `unknown`, এবং কখনো return না করা `never`।
